# S7 — 回测引擎

- 回测只接受已生成的 candidate signals，不负责策略审批。
- 信号时间点不能成交，入场统一在下一根 K 线开盘，防止 close 信号同 bar 成交。
- 历史成交按 `fills.yaml.backtest_historical` 加半点差、保守 buffer、线性冲击成本。
- 仓位按 stop distance 反推，并套用单标的名义上限、组合上限和 min-notional 地板。
- 费用按 order type 绑定 maker/taker，并应用 BNB discount 配置。
- 当前最小闭环以 time stop 或下一根 bar close 退出，stop 被触及时按 stop 出场。
- lookahead guard 对信号生成器做全量 vs 前缀对比，泄漏未来会被拒绝。
- S7 不调用交易所、不读取密钥、不执行订单。
