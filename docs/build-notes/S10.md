# S10 — 风控引擎

- 风控引擎输出 `RiskDecision`，不直接执行撤单或平仓。
- 仓位规模独立实现，按 stop distance 反推并套用单标的/组合/min-notional 约束。
- PnL circuit breaker 覆盖连亏 cooldown、滚动 24h、总回撤、业务硬止损。
- 运行态 circuit breaker 覆盖 data gap、API error、heartbeat miss、price deviation。
- kill switch 输出 cancel/halt/flatten 动作，供后续 broker runtime 消费。
- 阶梯顺序测试确保 3 笔满额亏损先触发 cooldown，4 笔才触发滚动 24h。
- S10 不修改 `risk-policy.yaml`，不削弱任何红线。
- S10 不连接交易所、不执行真实撤单或平仓。
