# S2 — 数据层：历史 OHLCV

- OHLCV 拉取封装为 `OhlcvClient` 协议，真实 ccxt 与测试 fake 共用同一接口。
- 分页逻辑按 `since` 推进，按 timestamp 去重，避免重复页造成重复 K 线。
- 历史数据仅使用公开 REST，不需要密钥，不触发任何交易权限。
- Parquet 写入根目录 `/data/raw/ohlcv`，DuckDB 写入 `/data/processed/market.duckdb`。
- 数据质量报告包含 bars、起止时间、重复 timestamp、缺口数、覆盖率。
- CLI 默认读取 `config/`，按 research symbols 和 `min_lookback_days` 拉取。
- 单元测试用 fake exchange，覆盖分页、去重、Parquet、DuckDB、质量报告。
- 当前网络位置 Binance 公共 API 返回 451，S2 使用 OKX 公共 REST 完成真实落库。
