# S4 — 盘口与资金费率采集器（前向）

- S4 只实现前向采集框架与落库，不启动默认外部长连接。
- 盘口数据用 `OrderbookSnapshot` + level 表落 DuckDB，供后续 paper 撮合。
- 采集器依赖 `OrderbookSource` 协议，测试用 fake source 模拟 websocket。
- source 断线抛异常时，collector 按上限重连并继续累计 snapshot。
- 资金费率用独立 `FundingRate` 模型，字段显式标记 `market_type=perp_only`。
- 文档 `docs/forward-market-data.md` 明确：L2 盘口只能从上线后前向积累，不能历史回填。
- S4 不读取密钥、不调用交易所私有端点、不启用对外告警或长跑任务。
- 真实 websocket 适配器留给后续调度/部署配置显式开启。
