# S6 — 策略 SDK 与两个策略

- 新增 `Signal` 数据结构，策略只输出候选信号，不做审批、仓位或执行。
- 新增 `StrategyState`，先保留最小状态接口，后续回测/执行可扩展。
- 动量策略使用 fast/slow MA、Donchian previous high、ATR stop，订单类型来自配置。
- 均值回归策略使用 RSI、Bollinger lower band、趋势过滤、ATR hard stop、time stop。
- 所有参数均来自 `strategies.yaml` / `risk-policy.yaml` 加载后的 Pydantic 配置。
- 策略对 warmup `None` 显式跳过，不隐式补 0。
- 测试覆盖同输入同参数复现、动量信号、均值回归止损与 time stop。
- S6 不批准策略，不进入 Signal Loop；审批与注册留给 S9。
